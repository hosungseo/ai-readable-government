Two source layers

보도자료는 설명이고,
관보는 공식 기록입니다.

이 사이트는 정부가 무엇을 설명했는지와 정부가 무엇을 공식적으로 기록했는지를 나란히 읽게 하는 인터페이스를 목표로 합니다.

Layer 01

Press

정책브리핑 보도자료를 날짜별·부처별로 읽고, 설명성 높은 readable markdown 아카이브로 탐색합니다.

  • 날짜별 아카이브
  • 부처별 인덱스
  • 원문 링크와 정책브리핑 source 연결

Layer 02

Gazette

관보 메타데이터, 원문 PDF, raw 추출, 후처리 readable-final까지 이어서 보는 공식 기록 레이어입니다.

  • 관보 일반 API 기반 메타데이터
  • PDF 캐시 및 raw markdown/json
  • 후처리된 readable-final
